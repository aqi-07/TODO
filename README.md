# TaskFlow

TaskFlow is an advanced, aesthetically pleasing Todo Application built with React. It provides a robust set of features to help you manage your daily tasks efficiently.

## Features

- **Add Tasks**: Quickly add new tasks.
- **Edit & Delete**: Modify existing tasks inline or remove them.
- **Mark Complete**: Keep track of what's done.
- **Search & Filter**: Find tasks easily and filter them by All, Completed, or Pending.
- **Persistent Storage**: Tasks are automatically saved to your browser's local storage so you never lose them.

## Getting Started

1. Clone the repository.
2. Run `npm install` to install the dependencies.
3. Run `npm run dev` to start the development server.
4. Open the provided local URL in your browser to view the app.
